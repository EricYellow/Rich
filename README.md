![demo](./Dec-09-2017 22-49-08.gif)



## Requirements
>=iOS8 , swift4

## Installation

Rich is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'Rich'
```
